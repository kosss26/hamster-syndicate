# Админ-панель «Путешествие знаний»

Планируется SPA на базе Vue 3 + Vite. На данном этапе зафиксированы требования:

- Управление категориями, вопросами и сюжетными главами.
- Просмотр статистики игроков, рейтингов, достижений.
- Настройка подсказок и сюжетных переходов.
- Аутентификация администраторов и аудит действий.

Структура и начальная реализация будут добавлены на следующих этапах разработки.
